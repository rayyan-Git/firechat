Step-by-Step Deployment Guide (100% Free)

1

Create Firebase Project

Firebase provides free backend services for your chat app.

  • Go to firebase.google.com
  • Click "Get Started" and create a new project
  • Name your project (e.g., "FireChat")
  • Disable Google Analytics for this demo
2

Configure Firebase Services

Set up the necessary services for your chat app:

  • In Firebase Console, go to Authentication > Sign-in method > Enable Google
  • Go to Realtime Database > Create Database > Start in test mode
  • No need to change security rules for this demo
3

Get Firebase Config

Add a web app to your Firebase project to get configuration:

  • In Firebase Console, go to Project Settings
  • Under "Your apps", click the web icon ()
  • Register your app (e.g., "FireChat")
  • Copy the configuration object

4

Create GitHub Repository

GitHub Pages provides free hosting for your chat app.

  • Go to github.com and log in
  • Click "+" > New repository
  • Name your repository (e.g., "firechat")
  • Set to "Public"
  • Click "Create repository"
5

Create HTML File

Create an index.html file with your chat app code:

  • Copy the entire code from this page
  • Replace the Firebase config placeholder with your actual config
  • Save the file as index.html

Important: Replace the placeholder config with your actual Firebase config!

6

Deploy to GitHub Pages

Upload your file and enable GitHub Pages:

  • In your GitHub repository, click "Add file" > "Upload files"
  • Upload your index.html file
  • Go to Settings > Pages
  • Set source to "main" branch and "/ (root)" folder
  • Click "Save"

Your app will be live at:
https://[username].github.io/[repository]
