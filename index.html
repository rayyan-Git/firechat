How to Set Up Your Own Firebase Project

Step 1: Create Firebase Project

Go to firebase.google.com and create a new project.

  • Click "Add project"
  • Name your project (e.g., "FireChat")
  • Disable Google Analytics for this demo

Step 2: Configure Services

Enable these services in Firebase Console:

  • Authentication > Sign-in method > Enable Google
  • Realtime Database > Create Database > Start in test mode

Step 3: Get Firebase Config

Get your configuration from Firebase Console:
